# STIsim documentation

STIsim is a stochastic agent-based model, written in Python, for exploring and analyzing sexually transmitted infections. The repository also contains scripts for analyzing model output and tests for testing functionality.

# STIsim

STIsim is an agent-based modeling framework in which users can design and configure simulations of sexually-transmitted diseases. STIsim uses the [Starsim](https://starsim.org) architecture, and belongs to the Starsim model suite which also includes [Covasim](https://covasim.org), [HPVsim](https://hpvsim.org), and [FPsim](https://fpsim.org).

## Installation

STIsim is most easily installed via PyPI: `pip install stisim`.

STIsim can also be installed locally. Clone this repository, then run `pip install -e .` (don't forget the dot at the end!).
